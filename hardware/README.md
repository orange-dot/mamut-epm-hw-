# Hardware Workspace

This directory holds hardware-targeted design work that has already passed basic concept and simulation stages.

Current intended layout:

- `p1-vco/` for the first oscillator prototype
- future voice and subsystem folders later

Good practice:

- only promote work here after the mechanism has been studied in `sim/`
- keep interface names and test points aligned with simulation notes
- keep bench evidence in `bench/`, not mixed into this area

Do not dump random bench files here.
This area should stay closer to intentional design artifacts.
